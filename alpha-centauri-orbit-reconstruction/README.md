# Alpha Centauri A B Orbital Reconstruction

This project numerically reconstructs the apparent orbit of Alpha Centauri B relative to Alpha Centauri A using published orbital elements.

The goal is to demonstrate computational methods in orbital dynamics and astrometric visualization using Python.

## Physics Background

The orbital motion is modeled as a Keplerian binary orbit characterized by:

- Orbital period
- Eccentricity
- Semi-major axis
- Inclination
- Longitude of ascending node
- Argument of periastron

The orbit is projected onto the plane of the sky to reproduce observable quantities such as:

- Angular separation
- Position angle

## Figures Produced

The code generates:

1. Apparent orbit of Alpha Centauri B relative to A
2. Angular separation vs time
3. Position angle vs time
